# Семестровая работа №2. Сетевое приложение с оконным интерфейсом.

## Краткое описание сетевой игры

Diamond Dungeons - игра для 3 игроков одновременно. Вы вместе исследуете подземелья, собираете алмазы, решаете головоломки и избегаете опасностей. Это не соревнование — вы команда, и побеждаете или проигрываете вместе.
